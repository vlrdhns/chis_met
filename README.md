### 1. Аналитический подход отделения корней нелинейного уравнения

Согласно пособию, процесс нахождения корней уравнения $f(x) = 0$ состоит из двух этапов. Первый этап — отделение корней, целью которого является выделение интервалов, содержащих только один корень.

Аналитический способ отделения корней базируется на двух основных теоремах математического анализа:

1. **Условие существования корня на отрезке:** Если непрерывная функция $f(x)$ на концах отрезка $[a, b]$ принимает значения разных знаков, то внутри этого отрезка содержится по меньшей мере один корень уравнения $f(x) = 0$. Формально это записывается так:
   
   $$f(a) \cdot f(b) < 0$$
   
   Рисунок 4.2 в пособии иллюстрирует этот случай: на отрезке может быть несколько корней.

2. **Условие единственности корня на отрезке:** Для того чтобы гарантировать, что на отрезке $[a, b]$ находится **только один** корень, к первому условию добавляется требование знакопостоянства первой и второй производных.
   - Функция должна принадлежать классу $C^2[a, b]$ (т.е. быть дважды непрерывно дифференцируемой).
   - Производные $f'(x)$ и $f''(x)$ должны быть непрерывны и сохранять свой знак на $[a, b]$.

Таким образом, аналитический подход заключается в исследовании функции: вычислении её значений на предполагаемых границах интервалов и анализе поведения её производных для выделения интервалов монотонности и неизменной кривизны графика (выпуклости/вогнутости), что гарантирует наличие ровно одного корня (рис. 4.1).

### 2. Метод простой итерации для решения систем двух нелинейных уравнений

В разделе 7.2 пособия описывается метод простой итерации для системы из двух нелинейных уравнений.

**Алгоритм метода:**

1. **Приведение к виду, удобному для итераций.** Исходная система

   $$f_1(x, y) = 0$$
   $$f_2(x, y) = 0$$

   преобразуется к эквивалентному виду:

   $$x = \varphi_1(x, y)$$
   $$y = \varphi_2(x, y)$$

2. **Построение итерационного процесса.** Задавшись начальным приближением $(x^0, y^0)$, последующие приближения вычисляются по единой формуле:

   $$x^{k+1} = \varphi_1(x^k, y^k)$$
   $$y^{k+1} = \varphi_2(x^k, y^k)$$

**Условия сходимости:**

Итерационный процесс сходится к единственному решению $(\xi, \eta)$ в области $R = \{a \le x \le b, c \le y \le d\}$, если выполнены следующие условия:

1. В области $R$ существует единственное решение.
2. Начальное приближение $(x^0, y^0)$ принадлежит области $R$.
3. В области $R$ функции $\varphi_1$ и $\varphi_2$ должны удовлетворять одному из условий сжатия. Пособие приводит два варианта достаточных условий (означающих, что отображение является сжимающим):
   
   - *Условие по строкам матрицы производных:*
     
     $$\left| \frac{\partial \varphi_1}{\partial x} \right| + \left| \frac{\partial \varphi_2}{\partial x} \right| \le q_1 < 1$$
     
     *или*
     
     $$\left| \frac{\partial \varphi_1}{\partial y} \right| + \left| \frac{\partial \varphi_2}{\partial y} \right| \le q_2 < 1$$
   
   - *Условие по столбцам матрицы производных:*
     
     $$\left| \frac{\partial \varphi_1}{\partial x} \right| + \left| \frac{\partial \varphi_1}{\partial y} \right| \le q_1 < 1$$
     
     $$\left| \frac{\partial \varphi_2}{\partial x} \right| + \left| \frac{\partial \varphi_2}{\partial y} \right| \le q_2 < 1$$

### 3. Численные методы решения задачи Коши

Восьмая тема пособия посвящена решению задачи Коши для обыкновенного дифференциального уравнения первого порядка $y' = f(x, y)$ с начальным условием $y(a) = y_0$ на отрезке $[a, b]$. Пособие описывает следующие группы методов:

1. **Метод численного интегрирования (стр. 47-48):**
   - Основан на замене исходной задачи эквивалентным интегральным уравнением Вольтерра.
   - Интеграл в правой части приближенно заменяется квадратурной формулой, что приводит к системе линейных алгебраических уравнений относительно значений искомой функции в узлах сетки.
   - Приводится пример для количества точек n=2 и n=5, где даны матрицы коэффициентов $A_j^{(i)}$.

2. **Метод Эйлера (стр. 48):**
   - **Простой метод Эйлера:** $y_{k+1} = y_k + h f(x_k, y_k)$. Имеет погрешность на шаге $O(h^2)$.
   - **Модифицированный метод Эйлера с уточнением (итерационный):** Сначала дается прогноз по простому методу, а затем он итерационно уточняется по формуле:
     
     $$y_{k+1}^{(i)} = y_k + \frac{h}{2} [f(x_k, y_k) + f(x_{k+1}, y_{k+1}^{(i-1)})]$$

3. **Методы Рунге-Кутта (стр. 49-50):**
   - Семейство методов, основанных на вычислении приближения к решению как взвешенной суммы значений функции $f(x, y)$ в нескольких промежуточных точках на шаге.
   - В пособии приведены формулы для методов 1-го, 2-го и наиболее распространенного 4-го порядка точности. Для метода 4-го порядка:
     
     $$y_{i+1} = y_i + \frac{1}{6}(k_1 + 2k_2 + 2k_3 + k_4)$$
     
     где $k_i$ вычисляются по формулам, приведенным на стр. 49.
   - Описан алгоритм выбора шага для достижения заданной точности $\varepsilon$ путем двойного просчета (с шагом $h_0$ и $h_0/2$).

4. **Метод Адамса (стр. 50):**
   - Многошаговый метод, который использует информацию не только в предыдущей, но и в нескольких предыдущих точках.
   - В пособии представлена схема "прогноза и коррекции" 4-го порядка, объединяющая явную и неявную формулы Адамса.

5. **Метод Милна (стр. 50):**
   - Еще один метод типа "прогноз-коррекция". Сначала по четырем предыдущим точкам дается прогноз значения $y_i^{\text{пpeq}}$, затем он корректируется по формуле, использующей два предыдущих значения и прогноз.

### 4. Применение метода прогонки с аппроксимацией граничных условий с первым порядком при решении линейных краевых задач

Вопрос касается краевых задач для ОДУ 2-го порядка, однако в основном разделе 8.6 пособия рассматриваются только граничные условия I рода ($u(a)=\mu_1$, $u(b)=\mu_2$). Дополнительный материал по этому вопросу можно найти в разделе **9.3.3**, где метод прогонки применяется к неявной схеме для **уравнения в частных производных параболического типа**.

В этом контексте:
- На каждом временном слое $t = t_{n+1}$ решается краевая задача для ОДУ 2-го порядка, аппроксимирующая исходное уравнение.
- В этой задаче заданы граничные условия I рода: $U(0, t) = \mu_1(t)$ и $U(1, t) = \mu_2(t)$.

**Применение метода прогонки с аппроксимацией ГУ 1-го порядка:**

Когда речь идет об аппроксимации граничных условий **с первым порядком**, это означает, что погрешность аппроксимации на границе имеет порядок $O(h)$, где $h$ — шаг по пространству. В данном случае, условия I рода задаются точно в узлах сетки $U_0^{n+1} = \mu_1(t_{n+1})$ и $U_M^{n+1} = \mu_2(t_{n+1})$, что является аппроксимацией с бесконечным порядком, так как значение задается точно, а не через производные.

Однако, метод прогонки, описанный в пособии, автоматически учитывает эти граничные условия.

1. **Аппроксимация краевой задачи.** После дискретизации по пространству и времени получается система линейных алгебраических уравнений вида (9.10):
   
   $$\sigma U_{m+1}^{1} - (1+2\sigma) U_{m}^{1} + \sigma U_{m-1}^{1} = -\tau \phi_m^0 - U_m^0$$
   
   которая решается для нахождения значений на первом слое $U_m^1$.

2. **Прямой ход прогонки.** Ищется связь между соседними значениями в виде:
   
   $$U_i = \alpha_i U_{i+1} + \beta_i, \quad i=0,1,\dots,M-1$$
   
   Чтобы эта формула работала и на левой границе, коэффициенты $\alpha_0$ и $\beta_0$ выбираются так, чтобы она удовлетворяла левому граничному условию $U_0 = \mu_1(\tau)$. Для этого полагают:
   
   $$\alpha_0 = 0, \quad \beta_0 = \mu_1(\tau)$$
   
   Подстановка $\alpha_0=0$ в уравнение связи дает $U_0 = 0 \cdot U_1 + \mu_1(\tau)$, что в точности соответствует левому граничному условию. Остальные прогоночные коэффициенты ($\alpha_m, \beta_m$) вычисляются по формулам (9.12).

3. **Обратный ход прогонки.** На правой границе известно $U_M = \mu_2(\tau)$. Используя полученные на прямом ходе коэффициенты, последовательно находятся все $U_i$ для $i=M-1, M-2, \dots, 1$ по формуле $U_i = \alpha_i U_{i+1} + \beta_i$.

Таким образом, задав $\alpha_0=0$, $\beta_0=\mu_1(\tau)$, мы обеспечиваем выполнение левого граничного условия **точно**, а не с первым порядком. Если бы требовалась аппроксимация с первым порядком, например, для условия вида $u'(a) = \sigma$, то начальные прогоночные коэффициенты выводились бы из разностной аппроксимации производной (например, $(U_1 - U_0)/h = \sigma$), что дало бы связь $U_0 = U_1 - \sigma h$ и, соответственно, $\alpha_0=1$, $\beta_0=-\sigma h$. В рассмотренном же в пособии случае условия I рода задаются точно.

### 5. Неявная разностная схема для решения уравнения теплопроводности

В разделе 9.3.2 пособия подробно описана неявная разностная схема для решения смешанной задачи для уравнения теплопроводности $\frac{\partial U}{\partial t} = \frac{\partial^2 U}{\partial x^2} + \phi(x, t)$.

**Определение схемы:**

Неявная схема (9.9) строится на шаблоне, включающем точки $(m-1, n+1)$, $(m, n+1)$, $(m+1, n+1)$ для пространственной производной и точку $(m, n)$ для временной производной. Это означает, что пространственная производная берется на **новом**, $(n+1)$-ом слое по времени.

Формула схемы имеет вид:

$$\frac{U_m^{n+1} - U_m^n}{\tau} = \frac{U_{m+1}^{n+1} - 2U_m^{n+1} + U_{m-1}^{n+1}}{h^2} + \phi_m^n$$

где:
- $U_m^n$ — значение функции в узле $(x_m, t_n)$,
- $\tau$ — шаг по времени,
- $h$ — шаг по пространству,
- $\phi_m^n$ — значение функции источника $\phi(x, t)$ в узле $(x_m, t_n)$.

**Свойства схемы (согласно пособию):**

1. **Порядок аппроксимации:** Схема аппроксимирует исходное дифференциальное уравнение с погрешностью $O(\tau + h^2)$, т.е. имеет первый порядок по времени и второй по пространству.
2. **Устойчивость:** Схема является **абсолютно устойчивой**. Это означает, что в отличие от явной схемы, здесь нет ограничения на соотношение шагов $\tau$ и $h$.
3. **Сходимость:** Благодаря аппроксимации и устойчивости, схема является сходящейся.

**Реализация (метод прогонки):**

Из-за неявности, для нахождения значений $U_m^{n+1}$ на новом слое необходимо решать систему линейных алгебраических уравнений. Как показано в п. 9.3.3, эта система имеет трехдиагональный вид:

$$\sigma U_{m+1}^{n+1} - (1+2\sigma) U_{m}^{n+1} + \sigma U_{m-1}^{n+1} = -U_m^n - \tau \phi_m^n$$

где $\sigma = \tau / h^2$. Такая система эффективно решается методом скалярной прогонки (алгоритм подробно описан в п. 9.3.3).
